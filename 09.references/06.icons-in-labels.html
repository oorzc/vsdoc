                      # 标签中的图标

                      你可以在插件中使用github提供的开源图标Octiconsopen in new window，你甚至可以在StatusBarItemopen in new window文本和QuickPickItemopen in new window标签中使用。添加图标的语法如下：

                      $(alert);

                      你还可以像这样使用多个标签

                      $(eye) $(heart) $(mark-github) GitHub
